# Heart Disease Classification (Machine Learning Project)

## Overview
This project uses machine learning to predict the presence of heart disease based on patient health data. The goal is to build and evaluate classification models using a real-world dataset.

---

## Dataset
This project uses the Heart Disease dataset from the UCI Machine Learning Repository:

Janosi, A., Steinbrunn, W., Pfisterer, M., & Detrano, R. (1989).  
Heart Disease Dataset. https://doi.org/10.24432/C52P4X

Note: The dataset is not included in this repository due to size and GitHub storage best practices.

---

## Project Workflow

1. Data Loading  
   - Dataset loaded directly from UCI repository

2. Data Preprocessing  
   - Handling missing values  
   - Feature selection and cleaning  
   - Data formatting for model training  

3. Model Development  
   - Supervised classification models used to predict heart disease  

4. Model Evaluation  
   - Accuracy  
   - Precision  
   - Recall  
   - F1-score  
   - Confusion Matrix  

---

## Results
The models were evaluated using standard classification metrics. Performance analysis highlights the strengths and limitations of each model in predicting heart disease.
